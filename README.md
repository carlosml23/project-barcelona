# project-barcelona

> Hackathon project — 24h build.

## Tech Stack

- **Language:** TypeScript
- **Runtime:** Node.js 20+
- **Package manager:** npm

## Quickstart

```bash
npm install
npm run dev
```

## Scripts

| Command | Purpose |
|---|---|
| `npm run dev` | Run in watch mode |
| `npm run build` | Compile to `dist/` |
| `npm run typecheck` | Type-check without emitting |

## Environment

Copy `.env.example` to `.env` and fill in values.
